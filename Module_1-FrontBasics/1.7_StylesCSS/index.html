Box Models, Display e Interfaces - Exercicios

O Box Model é um conceito central no CSS que descreve a estrutura e o espaçamento de elementos HTML. Cada elemento é representado como uma caixa retangular, composta por quatro partes principais, o content, o padding, o margin e o border

    Exercicíos
  • 1. Crie uma div com uma largura de 300px, altura de 150px, padding de 20px, borda de 5px e margem de 10px.
  • 2. Adicione uma cor de fundo azul a uma div e uma imagem de fundo com repetição desativada.
  • 3. Crie um layout básico usando display: block , display: inline e display: inlineblock
